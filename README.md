# Global Store Data Analysis Dashboard 📊

## 🧾 Project Overview

This project is an interactive **Power BI Dashboard** created to analyze Global Store sales data and generate meaningful business insights.

The dashboard helps in understanding:

* Overall sales and profit performance
* Regional sales analysis
* Category and sub-category performance
* Customer and segment insights
* Monthly sales trends
* Top-performing products
* Shipping and order analysis

The main goal of this project is to transform raw business data into actionable insights using data visualization techniques.

---

# 🎯 Objectives

* Analyze overall business performance
* Identify top profitable categories and products
* Track sales trends over time
* Compare regional performance
* Improve decision-making using data-driven insights
